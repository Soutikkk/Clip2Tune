# 🎵 Clip2Tune: MP4 to MP3 Converter 🎥

Clip2Tune is a sleek, simple, and powerful web application built with **Flask** and **MoviePy** that allows users to seamlessly extract high-quality audio from MP4 video files. 🚀

![Python](https://img.shields.io/badge/Python-3.8+-blue?style=for-the-badge&logo=python&logoColor=white)
![Flask](https://img.shields.io/badge/Flask-2.0+-black?style=for-the-badge&logo=flask&logoColor=white)
![MoviePy](https://img.shields.io/badge/MoviePy-Library-green?style=for-the-badge)

---

## ✨ Features

- 📂 **Drag & Drop**: Modern upload interface for easy file selection.
- ⚡ **Fast Conversion**: Extract audio from your videos in seconds.
- 🎨 **Minimalist Design**: A clean, aesthetic UI with smooth animations.
- 📥 **Instant Download**: Get your converted MP3 file immediately after processing.
- 🛠️ **Error Handling**: Friendly alerts if something goes wrong.

---

## 🛠️ Tech Stack

- **Backend**: [Flask](https://flask.palletsprojects.com/) (Python)
- **Audio Extraction**: [MoviePy](https://zulko.github.io/moviepy/)
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Font**: [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### 📋 Prerequisites

Ensure you have Python installed. You can download it from [python.org](https://www.python.org/).

### 🔧 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Soutikkk/Clip2Tune.git
   cd Clip2Tune
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   python app.py
   ```

4. **Open your browser**:
   Navigate to `http://127.0.0.1:5000` to start converting! 🎧
